First open folder 'flask_app' in Visual Studio Code then open terminal to install Libraries

1. You can install all the required libraries in a single line by running:

        pip install Flask opencv-python numpy

2. Now run app.py file.
3. Now Open browser and got to address:
    http://127.0.0.1:5000

4. Modify the H, S, and V values using the sliders shown in the video to detect specific colors and shapes. You need to adjust these values based on the desired color and shape.
